Introduction

Latent variable models

In a nutshell

Analyse multivariate data \(y=(y_1,\dots,y_p)^\top\) to measure and relate hidden variables \(\eta=(\eta_1,\dots,\eta_q)^\top\), \(q \ll p\), that explain complex patterns within the data.

In the social sciences, LVs are used to represent abstract constructs.

Photo credits: Unsplash @dtravisphd, @impulsq, @ev, @benmullins.

Latent variable models (cont.)

Measurement model

  • Define how manifest variables represent latent variables.
  • Also useful as a tool for dimensionality reduction.

Reduced form (aka marginal model)

  • The normal SEM can be written in reduced form as \(y_i \,\overset{\text{iid}}{\sim}\,\operatorname{N}_p(\mu, \Sigma)\), where \[\begin{gather} \mu := \mu(\kappa) = \nu + \cancelto{0}{\Lambda\alpha} \\ \Sigma := \Sigma(\kappa) = \Lambda(I-B)^{-1}\Psi(I-B)^{-\top} \Lambda^\top + \Theta \end{gather}\] as a function of model parameters \(\kappa\in\mathbb R^M\).

  • In the frequentist approach, estimates obtained by minimising discrepancy between \(S\) and \(\Sigma(\kappa)\) (Muth√©n and Satorra 1995), using

    • \(F_{ML}(\kappa) = \log |\Sigma(\kappa)| + \operatorname{tr}(S \Sigma(\kappa)^{-1}) - p\)
    • \(F_{GLS}(\kappa) = \operatorname{tr}[ (\Sigma(\kappa)-S)S^{-1} ]^2\)
    • \(F_{WLS}(\kappa) = (s-\sigma(\kappa))^\top W^{-1} (s-\sigma(\kappa))\)
    • Etc.

  • Fast and efficient, with nice asymptotic properties. See, e.g. {lavaan} (Rosseel 2012).

Conditional model

  • Alternatively, the normal SEM can be viewed as \[\begin{gather} y_i \mid \eta_i \sim \operatorname{N}_p(\nu + \Lambda \eta_i, \Theta)\\ \eta_i \sim \operatorname{N}_q\big(\cancelto{0}{(I-B)^{-1}\alpha}, (I-B)^{-1}\Psi(I-B)^{-\top} \big) \end{gather}\]

  • Adding suitable priors \(\kappa\sim f(\kappa)\) to the system, this form seems conducive for Bayesian estimation.

    • Many Bayesian SEM applications use Gibbs sampling (Congdon 2014; Lee 2007) available through software such as BUGS and JAGS.
    • Hamiltonian Monte Carlo (HMC) with Stan (Carpenter et al. 2017) may also be used. This is the default sampler in {blavaan} (Merkle and Rosseel 2018).
    • Lately, for efficiency, {blavaan} uses the marginal model instead, effectively turning SEM into covariance matrix model (Merkle et al. 2021).

Why Bayesian anyway?

  1. Flexibility
    • Move beyond standard assumptions (e.g., multivariate normality) to handle skewed distributions, non-normal errors, and outliers.
    • Enables robust handling of missing data.
  2. Extended inference and complex correlations
    • Support the estimation of full posterior distributions for LVs, allowing for extended inferences such as exceedance probabilities and inter-subject comparisons.
    • Model complex correlations, e.g. spatial or time-based dependencies across factors.
  3. + all other Bayesian good stuff

However, with it of course comes all the challenges: Slow convergence, label switching, sensitivity to priors, etc.

Integrated Nested Laplace Approximations

The SEM structure is also seemingly conducive for INLA (Rue, Martino, and Chopin 2009). Ingredients:

  1. Latent Gaussian model \([\eta\sim\operatorname{N}_q(0,\Psi)]\)
  2. Additive linear predictor \([\mu = \nu + \Lambda\eta + \cdots]\)
  3. Gaussian Markov Random Field \([Q=\operatorname{diag(\Sigma^{-1})}]\)
  4. Laplace approximation

formula = y ~ nu +
  f(y1, model = "iid") +
  f(y2, copy = "y1", hyper = list(beta = list(fixed = FALSE))) +
  f(y3, copy = "y1", hyper = list(beta = list(fixed = FALSE))) + ...

Case studies

Case 1: Multigroup SEM & measurement invariance

CFA model examining cognitive abilities in children from two different schools (Holzinger and Swineford 1939). Are constructs measured equivalently across groups?

  • Three types of measurement invariant models:

    1. Configural invariance
    2. Weak invariance: \(\Lambda_{g=1} \equiv \Lambda_{g=2}\)
    3. Strong invariance: \(\Lambda_{g=1} \equiv \Lambda_{g=2}\) and \(\nu_{g=1} \equiv \nu_{g=2}\)

  • Classically, these are formally tested using Likelihood Ratio Test (LRT) or Wald Test (in addition to comparing fit measures).

Bayesian model comparison

  • Under MCMC sampling schemes, we can only afford to perform approximate measurement invariant studies using informative prior distributions.

  • Using INLA, we can perform exact equality constraints and compare models.

    formula <- y ~ -1 + nu.g +

  ... +

Case 2: Multilevel SEM

  • Building a model based on procedural fairness theory, which links trust, legitimacy, and compliance with authorities (Jackson, Bradford, et al. 2011; Jackson, Hough, et al. 2011).

  • Often, country-level effects are ignored, but they can be crucial in understanding the variability in the latent constructs across countries.

Adding country-specific random effects

  • Let \(i=1,\dots,n_k\) index respondents, \(j=1,\dots,m\) index countries, and \(k=1,\dots,p\) index items. Additionally, let \(l=1,\dots,5\) index the latent factors.

  • Measurement model: \[\begin{align} y_{ijk} &= \lambda_{kl} (\eta_{il} + u_{jl}) + \varepsilon_{ijk} \\ &= \lambda_{kl} \eta_{il} + \tilde u_{jl} + \varepsilon_{ijk} \end{align}\] where the \(\tilde u_{jl}\) are Gaussian random effects. In {INLA}, we can specify this using an f(i, "iidkd") term.

  • Structural model, e.g.: \[\begin{align} \eta_{i3}^{(j)} &= \beta_{1}^{(j)} \eta_{i1}^{(j)} + \beta_{2}^{(j)} \eta_{i2}^{(j)} + \zeta_{i3}^{(j)} \\ \end{align}\] This is a bit challenging to specify in {INLA} (at the moment).

Case 3: Item Response Theory (IRT)

  • Suppose now observed data \(y_i = (y_{i1},\dots,y_{ip}) \in \{0,1\}^p\). Assuming each component \(y_{ik}\sim \operatorname{Bern}(\pi_{ik})\), we could model instead \[ \operatorname{logit}(\pi_{ik}) = \nu_k + \lambda_k z_i \] where \(z_i\,\overset{\text{iid}}{\sim}\,\operatorname{N}(0,1)\).

  • In the educational testing literature, this is so-called two parameter logistic (2PL) model1

    • \(\nu_k\) is the ‚Äúeasiness‚Äù parameter for item \(k\);
    • \(\lambda_k\) is the discrimination parameter for item \(k\); and
    • \(z_i\) is the unobserved latent trait for respondent \(i\).

  1. Albeit parameterised differently: Difficulty \(b_k \mapsto -\nu_k/\lambda_k\) and Discrimination \(a_k\mapsto \lambda_k\).

Spatial IRT

  • Traditional IRT analyses overlook regional disparities in educational outcomes, which can provide crucial insights for targeted interventions.
    • Increase funding, improving infrastructure, and teacher training in targeted areas.
    • Reveal socioeconomic factors that impact performance in specific regions.
  • A simple modification would be \[ \operatorname{logit}(\pi_{ijk}) = \nu_k + \lambda_k z_i + \phi_{j}, \] where \(\phi_{j}\) is a spatial random effect, e.g. (Besag, York, and Molli√© 1991): \[ \phi_j \mid \phi_{-j} \sim \operatorname{N}\Big(\rho \sum_{j'\sim j} w_{jj'}\phi_j, \ \tau^{-1}(1-\rho)^{-1}\Big), \hspace{2em} w_{jj'} = [j\sim j']. \]
  • In {INLA}, we make use of an f(region, model = "besagproper") term with family = "binomial" likelihood.

Conclusions

INLA for SEM

  • INLA is a promising alternative to MCMC sampling for üöÄ estimating Bayesian latent variable models and structural equation models.

  • Several gaps in {b/lavaan} that are easily filled by {INLA}:

    • Exact measurement invariance testing. üìè
    • Multilevel SEM. üèóÔ∏è
    • Spatial SEM. üåç
    • Non-normal likelihoods. üìä

  • A pivotal advancement in psychometric research, opening doors for innovative modelling of real-world data.

  • The ultimate goal is to provide an R software implementation for estimating Bayesian SEM using INLA.
